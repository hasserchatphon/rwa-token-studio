# Security Policy

This project is an educational prototype and should not be used for production investment activity.

## Reporting Issues

Open a GitHub issue for non-sensitive issues. For sensitive security findings, avoid public exploit details until a maintainer can review them.

## Current Limitations

- No real custody.
- No real KYC/AML provider integration.
- No audited smart contracts.
- No production transfer agent or broker-dealer workflow.
- No real fiat, stablecoin, or wallet settlement.
