# Contributing

Keep this project focused on educational and simulated tokenization flows.

## Local Checks

Run the full local gate before opening a pull request:

```bash
npm run validate
```

## Pull Request Checklist

- Keep regulated-finance claims conservative and accurate.
- Do not add secrets, keys, wallet seed phrases, or private URLs.
- Update `docs/` when changing compliance assumptions, routes, API contracts, or smart-contract behavior.
- Add or update tests when changing domain rules.
